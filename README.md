# Шаблон таблицы 

Шаблон таблицы предназначен для быстрого развертывания в react приложении. Выполнен при помощи библиотеки tanstack table




## Опции

1. Виртуализация строк и столбцов ✅
2. Поддержка группировки колонок
3. Закрепление шапки
4. Закрепление колонок
5. Ресайз колонок
6. DND перестановка колонок
7. Отключение колонок
8. Развертывание строк
9. Изменение данных в ячейках
10. Выбор строк
